# 🎨 Creative Market - Human-Made Art Marketplace

**Creative Market** is a premier digital marketplace dedicated 100% to human-made art, crafts, and design. We believe that true art is a profound language used to communicate human emotions, lived experiences, and the soul—a unique charm that no AI can ever replicate.

## ✨ Key Features

- **Fully Responsive UI:** Optimized for a seamless experience across Mobile, Tablet, and Desktop.
- **Artistic Aesthetic:** A unique design language featuring hand-drawn elements and a modern dark-themed interface.
- **Dynamic Layouts:** Custom CSS Skew techniques for artistic image slicing and interactive hover effects.
- **Performance Focused:** Built with **React + Vite** for lightning-fast load times and smooth transitions.

## 🛠️ Tech Stack

- **Frontend:** React.js
- **Styling:** Tailwind CSS (Modern Utility-First Framework)
- **Icons:** React Icons (FontAwesome & Lucide)
- **Typography:** Google Fonts integration (Anuphan, Rock Salt, Arizonia)
- **Deployment:** Vercel

## 📁 Project Structure Highlights

- `Market.jsx`: Showcases featured products with limited-edition badges.
- `ArtistLove.jsx`: High-impact section featuring tilted-grid layouts of top artists.
- `AboutUs.jsx`: Dedicated team section showcasing the creators behind the vision.
- `Footer.jsx`: Clean, social-integrated footer for better navigation.
